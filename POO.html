Programmation Orientée Objet (POO)

Qu'est-ce que la Programmation Orientée Objet (POO)?

La programmation orientée objet (POO) est un paradigme de programmation qui repose sur le concept d'objets. Les objets sont des instances de classes, et la POO permet de modéliser des entités du monde réel sous forme d'objets.

Principes de la POO

  • Encapsulation : Les données et les méthodes sont regroupées dans des classes et cachées à l'extérieur.
  • Héritage : Les classes peuvent hériter des propriétés et des méthodes d'autres classes.
  • Polymorphisme : Les objets peuvent prendre différentes formes et répondre de manière polyvalente aux appels de méthode.
  • Abstraction : Les détails complexes sont cachés pour simplifier la programmation.

Exemple de Code en POO

class Personne { constructor(nom, prenom) { this.nom = nom; this.prenom = prenom; } direBonjour() { console.log(`Bonjour, je m'appelle ${this.prenom} ${this.nom}`); } } const personne1 = new Personne('Doe', 'John'); personne1.direBonjour();
